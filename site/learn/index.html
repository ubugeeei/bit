// guide

Learning
Guide.

A guided path through bit. By the end you will have installed bit, made commits against a real repository, replicated history to a second peer, and opened a local pull request — all without leaving your terminal.

// 01

Install bit

One-line install on macOS & Linux, MoonBit toolchain, shell completion. The 30-second on-ramp.

~5 min
// 02

Your first commit

Initialize a repo, write a file, stage it, commit it, read the log. Git muscle memory works — bit is a drop-in.

~5 min
// 03

Going distributed

Clone over HTTP and over a bit relay. Push, fetch, and watch gossip converge two peers.

~10 min
// 04

PRs & issues without a server

bit pr and bit issue live as Git objects. Open a PR, review it, merge it — offline.

~10 min
// 05

Driving bit from MoonBit

Use the library API directly. Mount an in-memory TestFs, build a repository programmatically, run commands without a shell.

~5 min
